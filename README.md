# csoki-webshop

Csoki Webshop

Célunk egy letisztult weboldal létrehozása egy Kézmáves csokoládét árúsító cégnek. Célunk még, hogy a vásárlónak egyértelmű legyen a webshop felépítése, valamint gördülékeny legyen a vásárlás.

Erősségek • Reszponzivítás, működő kosár (fizetés még nem elérhető), oldalt közrefogó navigációs szekció 

Weboldal

Navbar: A cég logója, különbözző oldalak linkje, kosár ikon, mionden oldalon megtalálható. Fő oldal: egy bemutatkozó szöveg, melyben lényegre törően beszélnek a cégről. Termékek oldal: itt megtalálható a webshop összes terméke, melyekhez egyenként tartozik egy szöveg, valamint egy tovább és egy kosárba gomb. Egyes termékek oldalai: egy adott termék tovább gombjára kattintva érhetjük el a termék oldalát, melyen részletesebb információkat találhatunk róla. Kosár oldal: itt találhatóak a kosárhoz adott termékek melyeknek a mennyiségét is lehet állítani. Kapcsolat oldal: itt találhatóak a cég fontosabb adatai valamint hogy mikor van elérhető telefonos ügyfél szolgálat. Footer: minden oldalon megtalálható.

Feladatmegosztás

Soós Milán: végső ellenőrzés, Frontend fejlesztés, Backend fejlesztés, szöveg írása, képek készítése, UX design, tesztelés
Pál Gréti: Frontend fejlesztés, szöveg írása, képek készítése, UI/UX design

Technikai megvalósítás Frontend: HTML5, CSS, (van Bootstrap 5). CSS: projekt stílusok a style.css fájlban. JS könyvtárak: Bootstrap bundle + egyedi JS a kosárhoz.

Tesztelés, Reszponzív megjelenés: az oldalak mobilon, tableten és desktopon jól jelennek meg (Bootstrap rácsok helyesek). Böngésző kompatibilitás: Chrome, Firefox és Edge. • Linkek / CTA-k: minden belső link ás gomb működik (átirányítás az ajánlatkérő oldalra).

Fájlok • HTML oldalak: csipos-csoki.html, index.html, kapcsolat.html, kosar.html, levendula-mez-csoki.html, malna-pisztacia-csoki.html, narancs-mandula-csoki.html, sos-karamellizalt-mogyoro-csoki.html, termekek.html • CSS fájl: style.css • JS fájl: script.js •  Dokumentáció (ez a fájl): projekt összefoglaló + technikai leírás.
